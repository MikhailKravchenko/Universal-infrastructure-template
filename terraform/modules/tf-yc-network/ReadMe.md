# Yandex.Cloud Network Module

## Описание

Этот модуль создает базовую сеть в Yandex.Cloud: VPC и подсети по доступным зонам.

## Параметры

| Имя переменной | Тип    | Описание                          | Значение по умолчанию |
| -------------- | ------ | --------------------------------- | --------------------- |
| `zone`         | string | Зона, в которой создается подсеть | `"ru-central1-a"`     |

> Другие переменные не требуются — модуль сам создает сеть и подсети.

## Выводы

| Имя                  | Описание                                                  |
| -------------------- | --------------------------------------------------------- |
| `vpc_network_id`     | ID созданной VPC-сети                                     |
| `yandex_vpc_subnets` | Список подсетей по зонам в формате `{ zone = subnet_id }` |
| `available_subnets`  | Список всех доступных подсетей                            |

## Пример использования

```hcl
module "network" {
  source = "./modules/tf-yc-network"
}
```
